關於我

我是 Hugh Chen，一名以 React 為核心技術的前端工程師。擅長運用 React 19、Redux Toolkit 進行複雜狀態管理，熟悉 ECharts、Chart.js、Lightweight Charts 等多種資料視覺化方案。我喜歡將複雜的數據轉化為直覺的介面——讓資訊不再冰冷，而是真正被看見、被理解。

我的開發旅程從 HTML/CSS 的第一行程式碼開始，一年內累積超過 24 個 GitHub 專案，一步步走到能獨立完成全端應用。這段路讓我養成了扎實的工程思維：遇到問題就拆解它，不懂的技術就動手做一個專案來學。我重視程式碼品質與團隊溝通，相信好的產品來自清晰的思考。

  • 獨立開發全端數據平台，整合 3 套圖表引擎與即時資料 API
  • 精通 React 生態系：Redux Toolkit 狀態管理 + React Router v7 路由架構
  • 團隊協作經驗豐富，熟悉 Git 協作流程與模組化元件設計

精選作品

每個專案都是一次從想法到實作的完整旅程，展現在不同技術領域的探索與成長。以下是最能代表我技術能力的代表作品。

Haogoo（好股）— 全端股市分析平台

獨立開發的全端股市數據平台，採用 React 19 搭配 Redux Toolkit 管理複雜的跨元件狀態，整合 ECharts、Chart.js、Lightweight Charts 三套圖表引擎，實現 K 線圖、均線分析、產業板塊等專業金融視覺化功能。後端以 Express.js 建構 API 並部署至 Zeabur。

React 19 Redux ECharts Express.js

NeighborHub — 社群互動平台

多頁式 SPA 社群平台，運用 React Router DOM v7 實現完整路由架構，透過 Axios 串接後端 API 處理非同步資料流，以 Bootstrap 5 搭配 SCSS 客製化與 Swiper 輪播元件打造流暢的使用者介面與互動體驗。

React React Router Axios Bootstrap

Calendar Booking System — 預約管理系統

以純 JavaScript 開發的預約日曆系統，無任何框架依賴。實作完整 CRUD 操作、時段管理、可配置預約窗口與休息日管理，證明在框架之外同樣具備獨立解決複雜邏輯的能力。

JavaScript CRUD DOM 操作
